# Dashboard de KPIs Logísticos

Painel interativo de indicadores de desempenho logístico — com gráficos, tabela de transportadoras e comparativo mensal.

🔗 **[Acessar o dashboard](https://mgamamaschio.github.io/dashboard-kpis/)**

---

## Funcionalidades

- KPIs em destaque: total de entregas, taxa no prazo, taxa de atraso e prazo médio
- Comparativo com mês anterior (variação em % com indicador visual)
- Gráfico de barras com volume mensal separado por status
- Ranking de transportadoras por taxa de atraso
- Tabela completa de desempenho com badge de status (excelente / atenção / crítico)
- Filtro por período (troca todos os dados dinamicamente)

---

## Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, variáveis CSS)
- JavaScript puro
- [Chart.js](https://www.chartjs.org/) para os gráficos
- GitHub Pages para hospedagem

> Os dados exibidos são fictícios e têm fins exclusivamente demonstrativos.

## Sobre o projeto

Este dashboard foi inspirado no trabalho real de monitoramento de KPIs logísticos — acompanhamento de transportadoras, prazos e divergências — que faz parte da minha trajetória profissional. A ideia foi transformar essa experiência operacional em uma interface visual funcional construída com código.
